Quick R Tutorial

Frank Erickson

2016-08-22

Chapter 1 Introduction

This document covers the basics of using R, without going into discussion of statistical methodology. Chapters 2 and 4 will get you up to speed on navigating R. Chapter 3 covers common tasks: working with data and running optimization routines. The final chapter covers tables and graphs; my suggestions for programming; and various computational pitfalls.

Why R instead of Stata?

  • Of course, it’s free, so you can run your code now and into the future.
  • Objects are treated symmetrically, so we don’t have to use different rules for data tables, matrices, functions and variables. Nor do we have to learn new rules for when those objects are in Stata vs Mata or when data is loaded in memory vs on disk. Nor do we have to shuttle objects around between the various levels encountered in Stata.
  • While Stata restricts by-group operations to a select few, R effectively allows by g: {multiple lines of arbitrary code} if condition, as seen in 3.1.2.

1.1 Getting started with R

Go to the R project site and follow the instructions to download the latest version from CRAN, the Comprehensive R Archive Network.

After installation, I know of two good options for running R on Windows:

  • Hook up R to notepad++ with NppToR. This allows you to edit R scripts alongside all other documents. Commands are passed to the R console by customizable shortcuts and switching between multiple running consoles is easy.

  • Install the Rstudio development environment, which is restricted to one instance of R at a time and has a clumsy interface: an unhidable sidebar, and no drag-and-drop (of scripts into the editor, nor data into the console). On the other hand, it has handy “development” features that made compiling this tutorial a snap and would probably prove helpful if writing a package to contribute to CRAN. Rstudio also has the nice feature of not requiring administrator privileges for installation, which makes it a good choice for trying out R.

1.2 About this book

The book’s R code was compiled using R, maintained by the R Foundation; written in the Rstudio IDE; and typeset with Rmarkdown and bookdown by RStudio. Icons are from the “Very Basic. Android L Lollipop” set by Ivan Boyko.

Some notes give technical details and can be skipped the first time through.

Other notes warn about inconsistencies and pitfalls and should be read.

Exercises are interspersed and should be pretty quick to do.
